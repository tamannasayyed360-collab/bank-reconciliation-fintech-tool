# Bank Reconciliation Fintech Tool

A professional Bank Reconciliation Tool built using HTML, CSS, and JavaScript.  
This application simulates a CA firm-level reconciliation workflow with real-time calculations and a structured reconciliation statement.

Live Demo:  
https://tamannasayyed360-collab.github.io/bank-reconciliation-fintech-tool/

---

## Overview

This project calculates the Adjusted Bank Balance based on:

- Bank Balance as per Books  
- Deposits in Transit  
- Outstanding Cheques  
- Bank Charges  

The tool dynamically updates the reconciliation table as values are entered, providing an accurate adjusted balance instantly.

---

## Features

- Real-time adjusted balance calculation  
- Structured reconciliation table (CA-style format)  
- INR currency formatting  
- Dark mode / Light mode toggle  
- Reset functionality  
- Clean fintech-style user interface  
- Fully responsive layout  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

No external libraries or frameworks were used.

---

## How the Calculation Works

Adjusted Bank Balance is calculated as:

Adjusted Balance =  
Bank Balance (Books)  
+ Deposits in Transit  
- Outstanding Cheques  
- Bank Charges  

All values update dynamically as the user types.
